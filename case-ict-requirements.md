# ICT 行业大客户销售价值链业务需求说明

> **行业背景**：华某 CBG（消费者业务集团）ICT 产品线，面向政企大客户（如运营商、政府、金融、教育）销售企业级网络设备、智慧园区解决方案、云与计算产品。销售周期长（3–18 个月），客户决策链复杂，需要支撑售前、商务、交付、售后全流程协同。

---

## 1. 业务背景与挑战

| 问题 | 描述 |
|------|------|
| 线索转化率低 | 市场活动产生的线索无结构化跟进，丢失率超 40% |
| 商机管理混乱 | 销售跟进状态分散在个人表格，无法汇总预测 |
| 报价效率低 | 复杂方案报价依赖人工计算，平均耗时 3–5 天 |
| 客户关系浅 | 签单后无系统化经营，续约/增购依赖个人关系 |
| 交叉销售弱 | 缺乏客户全貌，产品线割裂，无法识别增购机会 |

---

## 2. 核心业务价值链

本次设计覆盖 ICT 大客户销售的完整价值链，聚焦以下 3 条主干：

### 2.1 从市场到线索（MTL — Market to Lead）

- 通过展会、数字营销、渠道合作等方式捕获目标客户意向
- 对潜客进行画像分析、线索评分与分配
- 线索孵化并转化为商机

**关键诉求：**
- 统一线索入口，支持多渠道汇聚（官网、展会、BD 手工录入）
- 线索评分模型（行业、规模、预算意向、互动行为）
- 线索分配规则引擎（按区域/产品线/客户级别）

---

### 2.2 从线索到收款（LTC — Lead to Cash）

从商机立项到合同履约，是 ICT 大客户销售的核心价值链。

#### 阶段 A：商机管理
- 商机立项（客户、产品线、预计金额、预计签单时间）
- 商机推进（需求确认、竞争分析、方案制定、客户汇报）
- 商机赢率分析与预测

#### 阶段 B：解决方案与报价
- 解决方案设计（标准化 BOM 选型 + 定制化方案）
- 报价单生成（含设备、服务、实施费用三段式报价）
- 报价审批（含特价申请、折扣审批、区域总监签批）

#### 阶段 C：合同管理
- 合同拟定（框架合同 / 项目合同 / 补充协议三类）
- 合同审批（法务、财务、业务多级审批）
- 合同签署与归档

#### 阶段 D：交付与回款
- 交付计划制定与里程碑跟踪
- 验收管理（分期验收）
- 回款计划与催款管理

**关键诉求：**
- 商机全生命周期状态机管理（从线索到签单的 8 个阶段门控）
- 标准化 BOM + 定制 BOM 双轨报价能力
- 特价审批流（与 ERP 对接）
- 合同三类模板管理（含电子签章集成）

---

### 2.3 客户成功管理（CSM — Customer Success Management）

签单后通过系统化经营提升客户健康度、降低流失、驱动增购/续约。

- **客户健康管理**：基于设备激活率、服务响应、使用率等指标构建健康评分
- **续约管理**：提前 180 天识别续约机会，制定续约方案
- **增购/交叉销售**：基于客户画像推荐产品组合
- **客户投诉与风险预警**：自动识别高风险客户，触发介入流程

**关键诉求：**
- 健康评分模型（可配置指标权重）
- 续约提醒与自动任务创建
- 客户 360 全景视图（关系图谱、互动记录、产品资产）

## 3. 核心业务角色

| 角色 | 职责 |
|------|------|
| 大客户经理（KAM） | 商机主责、客户关系经营、报价提交 |
| 售前解决方案工程师 | 方案设计、技术交流、BOM 选型 |
| 商务经理 | 报价审批、合同拟定、特价申请 |
| 区域销售总监 | 商机预测审核、重要折扣签批 |
| CSM 经理 | 客户成功跟进、健康评分管理、续约推进 |
| 财务 BP | 回款跟踪、合同财务审批 |

---

## 4. 关键业务对象（初步识别）

> *（供架构师在 BA 推导后验证，非最终定论）*

- 线索（Lead）
- 客户档案（Account）
- 联系人（Contact）
- 商机（Opportunity）
- 解决方案提案（Solution Proposal）
- 报价单（Quotation）
- 合同（Contract）
- 交付计划（Delivery Plan）
- 回款单（Payment Receipt）
- 健康评分（Health Score）
- 续约合同（Renewal Contract）
- 客户互动记录（Interaction Record）

---

## 5. 系统集成要求

| 集成系统 | 集成内容 |
|----------|---------|
| ERP（SAP） | 特价申请审批、合同履约、发货确认 |
| 电子签章平台 | 合同在线签署 |
| 呼叫中心（CC） | 客户服务工单同步 |
| 数据湖 | 客户行为数据、设备激活数据回流 |
| 营销自动化（MA） | 线索回流与评分联动 |

---

## 6. 非功能需求

- 用户规模：全国 2,000+ 销售/CSM 并发使用
- 数据量：客户档案 50 万+，历史商机 200 万+
- 响应时间：核心页面 < 2 秒，报表 < 5 秒
- 合规：支持数据本地化部署（政府客户要求）
